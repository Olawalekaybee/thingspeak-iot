# Contributing Guidelines

1. Fork the repository
2. Create a new branch
3. Make changes
4. Commit and push
5. Open a Pull Request

---

## Coding Style

* Use clean, readable C++
* Comment important logic
* Keep functions modular

---

## Reporting Issues

Use GitHub Issues and include:

* Description
* Steps to reproduce
* Expected behavior
